Installing

Use bower to download this extension with all required dependencies.

Depending on requirements you also need to include better-ajaxify-pushstate.js or better-ajaxify-hashchange.js. The first file implements havigation via HTML5 History API and this is preferred option for a website, because it's SEO-friendly. The second uses hash to indicate current state, so it could be useful for web applications.
